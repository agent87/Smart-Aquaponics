# Smart-Aquaponics

-Raspberry Pi 3 Model B
-Python 3.7 or later
-5 Volts Relay Module for Raspberry Pi to Controll Higher Voltage Electronics Devices
-As always an Editing environment; most probably Sublime Text for beginers
